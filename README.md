## вңЁ What Agenticana Does

### рҹ§  ReasoningBank вҖ” AI That Learns From Your Work
```bash
python scripts/reasoning_bank.py record \
  --task "Add Stripe webhooks" \
  --decision "Use raw body parser before JSON middleware" \
  --outcome "Webhook signature verified correctly" \
  --success true
# Next time: similarity 0.93 вҶ’ Fast Path вҶ’ 60% fewer tokens
```

### рҹ”Җ Swarm Dispatcher вҖ” 20 Agents in Parallel
```bash
python scripts/agentica_cli.py swarm .Agentica/swarm_manifest.json
# вҶ’ Runs backend-specialist + security-auditor + test-engineer simultaneously
```

### рҹҰ… Logic Simulacrum вҖ” AI Agents Debate Before Coding
```bash
python scripts/agentica_cli.py simulacrum "Should we use JWT or HttpOnly cookies?"
# вҶ’ backend-specialist, security-auditor, frontend-specialist debate in real-time
# вҶ’ Consensus reached вҶ’ Winner logs the rationale
# вҶ’ ADR document auto-generated
```

### рҹӣЎпёҸ Guardian Mode вҖ” Pre-Commit AI Gate
```bash
python scripts/guardian_mode.py install
# Every git commit is now intercepted:
#   [1] Sentinel audit     вҶ’ warns on issues
#   [2] Python lint        вҶ’ blocks on syntax errors
#   [3] Secret scan        вҶ’ blocks on hardcoded secrets
```

### рҹ“қ NL Swarm вҖ” Plain English to Swarm
```bash
python scripts/nl_swarm.py "Add authentication to Django, audit it, and write tests" --run
# вҶ’ Auto-detects agents: security-auditor, backend-specialist, test-engineer
# вҶ’ Generates manifest вҶ’ Dispatches swarm
```

### вң… Proof-of-Work Commits вҖ” Trusted Code
```bash
python scripts/pow_commit.py sign
# Trust Score: 85/100 (CERTIFIED)
#   Debate:      YES вҖ” session d1a3705f
#   Performance: OPTIMAL
#   Guardian:    PASSED
```

## рҹҡҖ Quick Start

### Prerequisites
- [Node.js 20+](https://nodejs.org)
- [Python 3.11+](https://python.org)
- [VS Code](https://code.visualstudio.com) with [GitHub Copilot](https://github.com/features/copilot)
- Git

### Option A: Clone & Setup
```bash
git clone https://github.com/ashrafmusa/AGENTICANA.git
cd Agenticana

# Windows
powershell -ExecutionPolicy Bypass -File setup.ps1

# Install MCP dependencies
cd mcp && npm install && cd ..
```

### Option B: Install Guardian (any project)
```bash
# Copy Agenticana into your project, then:
python scripts/guardian_mode.py install
# Every commit is now Secretary Bird approved рҹҰ…
```

### Activate MCP in VS Code
```json
// .vscode/mcp.json
{
  "servers": {
    "Agenticana": {
      "type": "stdio",
      "command": "node",
      "args": ["path/to/AGENTICANA/mcp/server.js"]
    }
  }
}
```

Then in Copilot Chat:
```
@backend-specialist build a REST API with auth
@security-auditor review this auth flow
@orchestrator plan the full payment system
```
